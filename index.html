    This is a repository to share ongoing work building animated HIV cohort data visualizations.

    Project maintained by CCASANET Hosted on GitHub Pages — Theme by mattgraham

    dAtaViz

    This is the planned location of the data visualization scripts. These scripts take as input data that is formatted using a known convention in HIV research - the CCASAnet DTP which is based on HICDEP.

    These animated graphics are implemented using R statistical language and encoded using MEncoder. As the videos play, we make observations using voice-over technology.
